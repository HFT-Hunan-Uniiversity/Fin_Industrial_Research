## 目标
- 同时启动后端 FastAPI、前端 Node API 和静态网页服务，打开首页并完成一次报告生成，验证 PDF 下载与预览。

## 运行步骤
- 启动后端：运行 `python agent_analysis_project\api_server.py`（端口 `8000`）。
- 启动前端 API：运行 `node web\server\server.js`（端口 `8081`）。
- 启动静态网页：在 `web` 目录运行 `python -m http.server 8080`（页面通过 `http://localhost:8080/index.html` 访问）。

## 验证流程
- 打开 `http://localhost:8080/index.html`，选择“新能源汽车”，输入任意问题点击生成。
- 观察加载阶段提示（SSE），进入“查看报告”页；点击“下载”按钮或查看内嵌 PDF 预览（存在 `pdf_url` 时）。

## 说明
- 前端与后端联通通过 Node 代理，后端生成的 PDF 链接将被前端捕获并展示；当前仅“新能源汽车”板块有效。
- 如遇 PDF 中文显示不全，请在本机环境运行（已自动检测 Windows 中文字体）。
